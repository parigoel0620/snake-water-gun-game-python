# 🎮 Snake Water Gun Game (Python)

A fun and interactive **command-line game built using Python** based on the classic **Snake 🐍, Water 💧, Gun 🔫** concept (similar to Rock–Paper–Scissors).

In this game, the **player competes against the computer**, where the computer randomly selects Snake, Water, or Gun. The winner is decided based on the predefined game rules.

---

## 🕹️ Game Rules

- 🐍 **Snake drinks Water** → Snake wins  
- 💧 **Water drowns Gun** → Water wins  
- 🔫 **Gun kills Snake** → Gun wins  
- If both choices are the same → **It's a draw**
